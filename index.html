Google — Calendar & Gmail
Connect your Google account for Calendar and Gmail access.
Show setup steps
1
Create a Google Cloud project
console.cloud.google.com → New Project → name it "Junkyard Dog"
2
Enable Calendar API + Gmail API
APIs & Services → Library → search and enable both
3
Create OAuth credentials → Web application
Credentials → Create → OAuth client ID → Web application. Add this exact URL as an authorised redirect URI:
https://jyd.netlify.app/index.html
4
Add yourself as test user
OAuth consent screen → External → add your Gmail as test user → add scopes: calendar, gmail.readonly, gmail.send
5
Add Client ID + Secret to Netlify env vars
Netlify → Site settings → Environment variables → add GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET. Then redeploy.
Or paste an access token manually (OAuth Playground fallback)
Get a token from OAuth Playground and paste it below.
With Remember keys checked, JD saves it and auto-refreshes every hour via the Netlify function.

Show setup steps

Azure AD → App registrations → New registration
portal.azure.com → Azure Active Directory → App registrations → New registration → name "Junkyard Dog"
2
Set redirect URI (Single-page application)
https://login.microsoftonline.com/common/oauth2/nativeclient
3
Add delegated permissions
Mail.Read · Mail.Send · Calendars.ReadWrite · Chat.Read · Chat.ReadWrite · ChannelMessage.Read.All · User.Read
4
Grant admin consent, copy Tenant ID + Client ID
Click "Grant admin consent" then copy both IDs from the App registration Overview page.
